# InterMine Has input format


  

- Chado
- Ensembl
- GO gene association files
- GO ontology files
- InParanoid orthologs
- PSI XML (protein interactions)
- UniProt XML
- Uniprot
- \[\[FASTA\]\]
- \[\[GFF3\]\]
- and many others. Custom formats can be supported through the
  data-source framework.

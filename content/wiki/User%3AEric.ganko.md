# Eric Ganko

I'm a computational biologist with
<a href="http://www.syngentabiotech.com" class="external text"
rel="nofollow">Syngenta Biotechnology</a> in Research Triangle Park, NC,
working on a number of data integration issues. I've been with Syngenta
since the start of 2009; from 2004-2008 I was a SPIRE postdoc in Todd
Vision's lab at UNC, where I worked on several genome evolution projects
including the
<a href="http://www.mimulusevolution.org/" class="external text"
rel="nofollow">Mimulus genome</a>.

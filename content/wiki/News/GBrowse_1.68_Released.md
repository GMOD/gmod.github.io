# News/GBrowse 1.68 Released

Version 1.68 of [GBrowse](../GBrowse.1) is a stability release
which improves the documentation and installation process. It includes a
new script called `gbrowse_netinstall.pl`, which will semi-automatically
install GBrowse across the Internet on
[Windows](../GBrowse_Windows_HOWTO), [Mac
OSX](../GBrowse_MacOSX_HOWTO) and
<a href="../GBrowse_Linux_HOWTO" class="mw-redirect"
title="GBrowse Linux HOWTO">Linux</a> platforms.

*Posted to the [GMOD News](../GMOD_News) on 2007/04/17*

# <span id="Chado_and_interoperability" class="mw-headline">Chado and interoperability</span>

Presentation by Chris Mungall and Pinglei Zhou. Introduces the
[middleware](GMOD_Middleware "GMOD Middleware") problem and how
[XML](Glossary#XML "Glossary") based middleware such as
[XORT](XORT.1 "XORT") and [Chado XML](Chado_XML "Chado XML") can address
these issues.

**Note:** This presentation is also available in [PowerPoint
format](File:Mungall-050516.ppt "File:Mungall-050516.ppt").

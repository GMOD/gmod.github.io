<div id="mw-imagepage-content" class="mw-content-ltr" lang="en"
dir="ltr">

[JBrowse](JBrowse.1 "JBrowse") now supports custom tracks. A Perl module
(`ImageTrackRenderer.pm`) for creating user-drawn image tracks is now
available, based on the CPAN
<a href="http://search.cpan.org/perldoc?GD.pm" class="external text"
rel="nofollow">GD.pm</a> module. This shows the output of the provided
example program, `draw-basepair-track.pl`, that uses this module to draw
arcs over a sequence representing the base-pairing interactions of RNA
secondary structure.

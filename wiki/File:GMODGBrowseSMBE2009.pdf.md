# <span id="Using_GMOD_for_evolutionary_genomics_and_next_generation_sequence_data" class="mw-headline">Using GMOD for evolutionary genomics and next generation sequence data</span>

An overview of GMOD and a detailed discussion using
[GBrowse](GBrowse.1 "GBrowse") to visualize [next generation sequence
data](Next_Generation_Sequencing "Next Generation Sequencing") using the
Bio-SamTools SAMtools [GBrowse
adaptor](GBrowse_Adaptors "GBrowse Adaptors"). This was
[presented](Training_and_Outreach "Training and Outreach") by [Dave
Clements](User:Clements "User:Clements") at a <a
href="http://ccg.biology.uiowa.edu/smbe/symposia.php?action=view&amp;sym_ID=27"
class="external text" rel="nofollow">SMBE 2009 GMOD workshop</a>.

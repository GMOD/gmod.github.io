# <span id="Comparative_Genomics_Software_Packages_at_GMOD" class="mw-headline">Comparative Genomics Software Packages at GMOD</span>

Poster by [Scott Cain](User:Scott "User:Scott"), [David
Clements](User:Clements "User:Clements") and the GMOD Consortium.
Outlines the options available in GMOD for displaying
<a href="Synteny" class="mw-redirect" title="Synteny">synteny</a>/comparative
genomics. Describes [Apollo](Apollo.1 "Apollo"), [CMap](CMap.1 "CMap"),
[GBrowse_syn](GBrowse_syn.1 "GBrowse syn"), [Sybil](Sybil "Sybil"),
[SynBrowse](SynBrowse "SynBrowse"), and
<a href="Synview" class="mw-redirect" title="Synview">Synview</a>.

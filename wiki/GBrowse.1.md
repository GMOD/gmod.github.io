## <span id="About_Generic_Genome_Browser_.28GBrowse.29" class="mw-headline">About Generic Genome Browser (GBrowse)</span>

GBrowse is a combination of database and interactive web pages for
manipulating and displaying annotations on genomes. Features include:

- Simultaneous bird's eye and detailed views of the genome.
- Scroll, zoom, center.
- Use a variety of
  <a href="/wiki/GBrowse_Configuration_HOWTO#Glyphs_and_Glyph_Options"
  class="mw-redirect" title="GBrowse Configuration HOWTO">premade
  glyphs</a> or create your own.
- Attach arbitrary URLs to any annotation.
- Order and http://gmod.org/wiki/appearance of tracks are customizable
  by administrator and end-user.
- Search by annotation ID, name, orhttp://gmod.org/wiki/ comment.
- Supports third party annotation using [GFF](/wiki/GFF "GFF")
  formathttp://gmod.org/wiki/s.
- Settings persist across sessions.
- DNA and [GFF](/wiki/GFF "GFF") dumps.
- Connectivity to different databases, including [BioSQL
  and](/wiki/BioSQL "BioSQL")
  <a href="/wiki/Chado" class="mw-redirect" title="Chado">Chado</a>.
- Multi-lhttp://gmod.org/wiki/anguage support.
- Third-party feature loading.
- Customizable
  [plug-http://gmod.org/wiki/in](/wiki/GBrowse_Plugins "GBrowse Plugins")
  architecture (e.g. run
  <a href="http://en.wikipedia.org/wiki/BLAST" class="extiw"
  title="wp:BLAST">BLAST</a>, dump & import many formats, find
  oligonucleotides, [design
  primers](/wiki/PrimerDesigner.pm "PrimerDesigner.pm"), create
  restriction maps, edit features)

Note that the information on this page refers to GBrowse 2; GBrowse
1.http://gmod.org/wiki/x is recommended only for applications where
legacy browser support is required and a single database is used.

  
Visit the
<a href="http://gbrowse.http://gmod.org/wiki/org" class="external text"
rel="nofollow">GBrowse website</a>.

### <span id="System_Requirements" class="mw-headline">Systehttp://gmod.org/wiki/m Requirements</span>

GBrowse is [Perl](/wiki/Glossary#Perl "Glossary")-based and the GBrowse
2.x modules are
<a href="http://search.cpan.org/dist/GBrowse/" class="external text"
rel="nofollow">hosted on CPAN</a>. GBrowse can be installed using the
standard Perl module build procedure, or automated using a network-based
install script. In order to use the net installhttp://gmod.org/wiki/er,
you will need to have Perl 5.8.6 or higher and the Apache web server
installed.

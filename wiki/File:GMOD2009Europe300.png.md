<div id="mw-imagepage-content" class="mw-content-ltr" lang="en"
dir="ltr">

The logo for [GMOD Europe 2009](GMOD_Europe_2009 "GMOD Europe 2009"). a
week long event starting with the [2009 GMOD Summer School -
Europe](2009_GMOD_Summer_School_-_Europe "2009 GMOD Summer School - Europe")
and finishing with the [August 2009 GMOD
Meeting](August_2009_GMOD_Meeting "August 2009 GMOD Meeting").

This logo is also available in several formats:

- [503 x 430](File:GMOD2009Europe.png "File:GMOD2009Europe.png")
- **300 x 256**
- [170 x 145](File:GMOD2009Europe170.png "File:GMOD2009Europe170.png")
- <a href="../mediawiki/images/8/83/GMOD2009Europe.xcf" class="internal"
  title="GMOD2009Europe.xcf">Gimp image source file</a>

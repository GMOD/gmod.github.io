# <span id="Development_of_tools_for_the_analysis_and_visualisation_of_second_generation_sequencing_data_for_Brassica_species" class="mw-headline">Development of tools for the analysis and visualisation of second generation sequencing data for *Brassica* species</span>

Presentation by Chris Duran at [PAG 2010](PAG_2010 "PAG 2010"). The talk
covers three broad areas:

- *Brassica* gene and promoter discovery: TAGdb
- *Brassica* genome sequencing and annotation
- Linking genetic and genomic data using [CMap3D](CMap3D "CMap3D")

[CMap3D](CMap3D "CMap3D") is a comparative mapping tool that visualizes
correlations in three dimensions. It rests on top of a
[CMap](CMap.1 "CMap") database. See [CMap3D](CMap3D "CMap3D") for more.

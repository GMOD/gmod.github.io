Introduction

This document covers the specifics of Chado and how it is used within FlyBase.

Public Chado Access

FlyBase provides public access to our Chado database via a couple of methods. One method is to download our PostgreSQL dumps from ftp://ftp.flybase.net/releases/current/psql/ and install that on a local PostgreSQL server. Alternatively, you can directly connect to our PostgreSQL server which runs our current release. Information about how to connect to this database is detailed in Public Chado Databases.

Bio%253A%253AChado%253A%253ASchema

Bio%253A%253AChado%253A%253ASchema is a standard DBIx::Class layer for Chado, in Perl. DBIx::Class is an object-relational mapping system similar to other ORMs in other languages, such as Python’s SQLAlchemy and Java’s Hibernate.

If you use Chado and Perl, please consider contributing to the project. See the Development section below.

Bio%253A%253AChado%253A%253ASchema is a standard set of Perl modules that should work with any database supported by DBIx::Class. So far, it has been used successfully with Chado schemas running on both PostgreSQL and Oracle. Please try it out and report back to the mailing list!

Development

To-Do

  • Need many more many_to_many relationships throughout. These have to be added manually. If you find one missing that you need, please git clone the repository and send a pull request with it.
  • Need more compatibility with BioPerl. Right now you can use a Sequence::Feature as a Bio::SeqI, but that’s all. Patches and ideas are welcome, see the BCS discussion page.

News/GMOD Tutorials! GMOD Training!

Detailed and hands-on online tutorials are now available for the GMOD Components that were covered at the 2009 GMOD Summer Schools. Each tutorial includes a VMware system image to work with, and detailed instructions on how to setup the component on that image.

The tutorials are:

  • Apollo Tutorial - by Ed Lee
  • Artemis-Chado Integration Tutorial - by Robin Houston, Tim Carver and Giles Velarde
  • BioMart Tutorial - by Junjun Zhang and Syed Haider
  • Chado Tutorial - by Scott Cain, Joshua Orvis, and Dave Clements
  • GBrowse Tutorial - by Scott Cain and Sheldon McKay
  • GBrowse_syn Tutorial - by Sheldon McKay
  • JBrowse Tutorial - by Mitch Skinner and Ian Holmes
  • MAKER Tutorial - by Carson Holt
  • Tripal Tutorial - by Stephen Ficklin

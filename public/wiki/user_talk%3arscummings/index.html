2. For larger edits, considering making a new wiki page.

If you make a relatively large change (one requiring more than a paragraph of explanation), feel free to create a new wiki page for it. To maintain some degree of organization, please create this page under the JBrowseDev/Upcoming namespace. For example, a page you want to call ‘MyImprovedJBrowseFeature’ would be created as:

JBrowseDev/Upcoming/MyImprovedJBrowseFeature

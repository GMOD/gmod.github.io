Mimosa - Miniature Model Organism Sequence Aligner

Mimosa is a newly proposed GMOD component that is being developed on Github https://github.com/GMOD/mimosa

You can play around with an example Mimosa instance at http://testing.mimosa.sgn.cornell.edu .

The Mimosa README on Github has all the details about installing, using and developing Mimosa:

https://github.com/GMOD/mimosa/blob/master/README.md

Integration With GMOD Components

Mimosa builds on top of NCBI BLAST, BioPerl, Bio%253A%253AChado%253A%253ASchema and can optionally deploy into a standalone SQLite database, or an already-existing Chado database.

Mimosa on CPAN

You can view App::Mimosa documentation online at http://p3rl.org/App::Mimosa . To install from the command-line with CPAN.pm or cpanminus:

 cpan  App::Mimosa
 cpanm App::Mimosa

Mailing List

If you are interested in using Mimosa or helping develop it, you are invited to join our mailing list:

http://groups.google.com/group/gmod-mimosa

Bug Tracker

If you have found a bug or you would like to review our current tickets, check out our Github Issues:

https://github.com/GMOD/mimosa/issues

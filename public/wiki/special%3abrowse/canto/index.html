Has description Curate GO annotations for proteins using UniProtKB identifiers  + , Canto is an online tool that supports func … Canto is an online tool that supports functional gene annotation by community researchers as well as by professional curators. It provides a simple interface to gather genetic and molecular data from publications for inclusion in public biological databases. Canto employs OBO ontologies to ensure a consistent vocabulary is used to describe gene and gene product functionality and location. Genetic and physical interaction annotation is also supported. Canto captures all necessary information about each interaction for submission to BioGRID. Canto currently supports annotation export in JSON format, either by individual paper or in bulk.

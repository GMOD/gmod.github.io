Flash GViewer

Flash GViewer is a customizable Flash movie that can be easily inserted into a web page to display each chromosome in a genome along with the locations of individual features on the chromosomes. It is intended to provide an overview of the genomic locations of a specific set of features - eg. genes and QTLs associated with a specific phenotype, etc. rather than as a way to view all features on the genome. The features can hyperlink out to a detail page to enable to GViewer to be used as a navigation tool. In addition the bands on the chromosomes can link to definable URL and new region selection sliders can be used to select a specific chromosome region and then link out to a genome browser for higher resolution information.

Description

Chado%253A%253AAutoDBI is a middleware package based on Class::DBI and maps Perl objects directly to tables in the Chado schema. It provides a very easy Perl interface for access to the database. It is currently used as a bulk loader for the Chado database, and as the underlying Object-Relational Mapping (ORM) tool in Modware. Chado%253A%253AAutoDBI is automatically generated from the Chado database schema which makes it very easy to update when changes are made. This code autogeneration process was adapted from the Turnkey project which is a generic ORM/website code generation tool.

Java TreeView

Java TreeView, a portable Java application that has been tested on Linux, Windows, MacOS9 and MacOSX. Java TreeView is a visualization for various clustering algorithms. Commonly microarray data are clustered, for example with hierarchical clustering. Java TreeView provides an easy to use interface to explore clustered experiments involving large sets of experiments or genes. Java TreeView is not specific to microarray data and has been used to display the results of clustering sequence motifs.
